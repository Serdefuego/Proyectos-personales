CuentaGanado – Contador con JavaScript
 Descripción del proyecto

CuentaGanado es un proyecto simple desarrollado con HTML, CSS y JavaScript cuyo objetivo es practicar el manejo del DOM, eventos y lógica básica de programación mediante la creación de un contador interactivo.

El proyecto permite sumar y restar valores utilizando botones, mostrando el resultado en pantalla en tiempo real.

 Objetivo

El objetivo principal de este proyecto fue:

Practicar JavaScript desde lo básico

Entender cómo funciona un contador

Aprender a conectar JavaScript con HTML

Trabajar con eventos (click)

Separar lógica (JS) de estructura (HTML) y estilo (CSS)

 Qué hice en el proyecto

Creé la estructura HTML del contador

Mostré el valor inicial del contador en pantalla

Implementé botones para:

➕ Sumar

➖ Restar

Usé addEventListener para detectar clics

Actualicé el contenido del contador usando textContent

Apliqué estilos con CSS para centrar y presentar mejor la interfaz

Detecté y corregí errores como:

Duplicación de elementos

Uso incorrecto de id

Problemas de estructura HTML

 Tecnologías utilizadas

HTML5 → estructura del contenido

CSS3 → estilos y diseño visual

JavaScript → lógica del contador y eventos

 Aprendizajes clave

Un id debe ser único en todo el documento

El CSS no duplica elementos, solo los estiliza

El DOM se modifica desde JavaScript

Los eventos permiten interacción con el usuario

Es mejor empezar con proyectos simples antes de escalar

 Estado del proyecto

 Proyecto funcional y en mejora continua
